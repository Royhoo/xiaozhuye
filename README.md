
Application created by [ThinkJS](http://www.thinkjs.org)

Demo [小主页](http://xiaozhuye.com)

## Install dependencies

```
npm install
```

## Start server

```
npm start
```

## Deploy with pm2

Use pm2 to deploy app on production enviroment.

```
pm2 startOrReload pm2.json
```

## 前端js打包

```
npm run fe-xiaozhuye
npm run fe-blog
```

## 便捷更新生产环境

```
sh rebuild.sh
```
